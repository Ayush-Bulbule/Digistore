Step 1: Unzip the zipped file

Step 2: Open cmd and run npm install to install all the dependencies.

Step 3: Run npm run dev to run project locally. At localhost:3300

Step 4: Run npm run watch to run laravel-mix.

Step 5: Import the file products.json in mongo database > digitstore > collection > products

Step 6: Open project in browser try to register your self.

Step 7: To access admin functionality, Open mongodb > digitstore > collection > user >
        edit your role as admin. then logout and login again.
